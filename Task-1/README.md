🚢 Titanic Survival Prediction using Machine Learning
📌 Overview

The Titanic Survival Prediction project is a Machine Learning classification model that predicts whether a passenger survived the Titanic disaster or not.
The model analyzes important passenger features such as age, gender, passenger class, fare, and other attributes to determine the probability of survival.
This project demonstrates the complete Machine Learning workflow, including data preprocessing, feature engineering, model training, and evaluation.

🛠️ Technologies Used
🐍 Python
📊 Pandas
🔢 NumPy
🤖 Scikit-learn
☁️ Google Colab
⚙️ Project Workflow

The project follows a structured Machine Learning pipeline:
1️⃣ Data Loading and Exploration
Loaded the Titanic dataset and explored important features.

2️⃣ Handling Missing Values
Treated missing values in columns like Age and Embarked.

3️⃣ Data Cleaning and Preprocessing
Removed unnecessary columns and prepared the dataset for modeling.

4️⃣ Feature Encoding
Converted categorical variables (like Sex and Embarked) into numerical format.

5️⃣ Train-Test Split
Split the dataset into training and testing sets.

6️⃣ Model Training
Trained a Logistic Regression classification model.

7️⃣ Model Evaluation
Evaluated the model using accuracy score.

📈 Model Performance

✅ Algorithm Used: Logistic Regression
🎯 Model Accuracy: 79.8%

The model successfully predicts passenger survival based on historical Titanic dataset features.
